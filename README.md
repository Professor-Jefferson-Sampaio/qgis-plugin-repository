# QGIS Plugin Repository

Repositório de plugins para QGIS.

## Plugins disponíveis

### Food Map Points

Plugin que carrega pontos do site food-map.top e exibe no mapa.

## Instalação no QGIS

Adicionar repositório:

Plugins → Gerenciar Plugins → Repositórios

URL:

https://food-map.top/qgis_repo/plugins.xml.php
